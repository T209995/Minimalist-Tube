🧘 Minimalist Tube - Focus Mode

Minimalist Tube est une extension Chrome conçue pour transformer votre expérience YouTube en un outil de travail pur, en éliminant toutes les distractions algorithmiques.

✨ Fonctionnalités

Cette extension permet de masquer sélectivement les éléments suivants via un simple interrupteur :

Page d'accueil : Cache tout le flux de recommandations pour vous forcer à utiliser la barre de recherche.

Vidéos suggérées : Supprime la colonne latérale droite lors de la lecture d'une vidéo.

YouTube Shorts : Masque les sections et liens vers les Shorts pour éviter le "doomscrolling".

Commentaires : Cache la section commentaires pour éviter les distractions textuelles.

Écran final : Supprime les suggestions de vidéos à la fin d'une lecture.

Interface épurée : Centre automatiquement le lecteur vidéo pour une expérience cinématographique.

🚀 Installation (Mode Développeur)

Si vous souhaitez tester l'extension localement :

Téléchargez ou clonez ce dépôt

Ouvrez Chrome et accédez à chrome://extensions/.

Activez le Mode développeur (en haut à droite).

Cliquez sur Charger l'extension déballée.

Sélectionnez le dossier contenant les fichiers de l'extension.

🛠️ Technologies utilisées

JavaScript (Vanilla) : Logique de toggle et communication avec l'API chrome.storage.

CSS3 : Injection de styles dynamiques via des sélecteurs complexes pour manipuler le DOM de YouTube.

HTML5 : Interface utilisateur du popup.
